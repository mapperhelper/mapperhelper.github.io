MyBatis

通用 Mapper 3

@liuzh

通用Mapper都可以极大的方便开发人员。可以随意的按照自己的需要选择通用方法，还可以很方便的开发自己的通用方法。

极其方便的使用MyBatis单表的增删改查。

支持单表操作，不支持通用的多表联合查询。

支持 Mybatis-3.2.4 及以上版本

特别强调

  • 不是表中字段的属性必须加 @Transient 注解
  • 通用 Mapper 不支持 devtools 热加载，devtools 排除实体类包即可，配置方式参考：using-boot-devtools-customizing-classload
